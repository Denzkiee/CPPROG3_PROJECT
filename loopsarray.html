Loops in JavaScript

A loop allows you to repeat a block of code multiple times, which is useful when you need to perform repetitive tasks (like iterating over elements in an array, processing a set of numbers, etc.).

Types of Loops



a. For Loop

A for loop is used when you know how many times you want the loop to run.

Syntax:


        for (initialization; condition; increment/decrement) {
        // Code to run on each iteration
      }
  • Initialization: Initializes the counter (usually a variable).
  • Condition: The condition that must be true for the loop to keep running.
  • Increment/Decrement: This updates the counter (e.g., increase by 1 each time).

    • Example:

    
        for (let i = 0; i < 5; i++) {
        console.log("Iteration " + i);
      }

    Output:

    
        Iteration 0
        Iteration 1
        Iteration 2
        Iteration 3
        Iteration 4


    While Loop

    A while loop will execute a block of code as long as a specified condition is true. The difference from the for loop is that the while loop doesn't require you to specify initialization and update in one line 

    
    let i = 0;
    while (i < 5) {
        console.log(i);
        i++;  // increment the loop variable
    }

    OUTPUT:

    
      0
      1
      2
      3
      4


    Do while Loop

    A dowhile loop works similarly to the while loop, but it guarantees that the code will run at least once because the condition is checked after the first iteration.

    Syntax:

    
    do {
        // Code to be executed
    } while (condition);

    Example:

    
  let i = 0;
  do {
      console.log(i);
      i++;
  } while (i < 5);

    Output:

    
  0
  1
  2
  3
  4
  5


    What is an Array?

    An array is a special variable that can hold multiple values at once. Instead of declaring individual variables for each value, you can store multiple values in a single variable using an array. Arrays are ordered collections of values, which can be of any type (numbers, strings, booleans, objects, etc.). In JavaScript, arrays are zero-indexed, meaning the first item in an array is accessed at index 0.

    Creating an Array

    You can create an array in JavaScript using square brackets []. You can either define the array with predefined values or initialize it as an empty array.

    Example

    
      // An array with numbers
  let numbers = [1, 2, 3, 4, 5];
  
  // An array with strings
  let fruits = ["apple", "banana", "cherry"];
  
  // An empty array
  let emptyArray = [];

    Accessing Elements in an Array

    You can access elements in an array using their index. Remember, the index starts from 0.

    Example:

    
    let fruits = ["apple", "banana", "cherry"];
    console.log(fruits[0]);  // Output: "apple"
    console.log(fruits[1]);  // Output: "banana"
    console.log(fruits[2]);  // Output: "cherry"


    Modifying Array Elements

    You can modify an array element by assigning a new value to an existing index.

    
      let fruits = ["apple", "banana", "cherry"];
  
      fruits[1] = "blueberry";  // Changing "banana" to "blueberry"
  
      console.log(fruits);  // Output: ["apple", "blueberry", "cherry"]


    Array Length

    To get the number of elements in an array, use the .length property. This will return the total count of elements in the array.

    
    let fruits = ["apple", "banana", "cherry"];
  
    console.log(fruits.length);  // Output: 3


    Other Useful Array Methods

    JavaScript provides many built-in methods for manipulating arrays. Here are some common ones:map()

    The map() method creates a new array by applying a function to each element of the original array.

    
      let numbers = [1, 2, 3];
      let doubled = numbers.map(function(number) {
          return number * 2;
      });
  
      console.log(doubled);  // Output: [2, 4, 6]
