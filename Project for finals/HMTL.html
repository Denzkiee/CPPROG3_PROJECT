Lesson 1: Introduction to HTML

What is HTML?
HTML (HyperText Markup Language) is the standard markup language used to create web pages. It structures the content and layout of a website, using tags to define elements like headings, paragraphs, links, and images.

Basic Structure of an HTML Document:
The basic structure of any HTML document starts with a doctype declaration (<!DOCTYPE html>) followed by the opening <html> tag and two main sections: the <head> and <body>.

Example:

This is a basic HTML page with a heading and a paragraph:

    <!DOCTYPE html>
    <html lang="en">
      <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Sample Page</title>
      </head>
      <body>
        <h1>Welcome to My First HTML Page</h1>
        <p>This is my very first webpage built with HTML!</p>
      </body>
    </html>

Result:

Welcome to My First HTML Page

This is my very first webpage built with HTML!

Lesson 2: HTML Elements and Tags

What are HTML Tags?
HTML tags define the elements of the document. Most HTML tags come in pairs: an opening tag (e.g., <h1>) and a closing tag (e.g., </h1>), which encloses the content.

Common HTML Tags:

  • <p>: Defines a paragraph.
  • <h1> to <h6>: Defines headers.
  • <a>: Creates hyperlinks.
  • <img>: Embeds images.

Example:

This is a simple paragraph, a header, a link, and an image:

    <p>This is a paragraph of text.</p>
    <h1>Main Heading</h1>
    <a href="https://www.example.com">Click here</a>
    <img src="image.jpg" alt="An example image" width="500" height="300">

Result:

This is a paragraph of text.

Main Heading

Click here

Lesson 4: Links and Images

Links in HTML:
The <a> tag is used to define hyperlinks. You specify the destination URL in the href attribute.

Example (Link):

Visit Example 
    <a href="https://www.example.com">Visit Example</a>

Images in HTML:
The <img> tag is used to display images on the page. The src attribute specifies the image URL, and the alt attribute provides alternative text for accessibility purposes.

Example (Image):

    <img src="https://via.placeholder.com/150" alt="Placeholder Image" width="150" height="150">

Lesson 5: Tables

Creating Tables in HTML:
HTML tables are used to display tabular data. They are made up of several elements: <table>, <tr> (table row), <th> (table header), and <td> (table data).

Example (Table):

Header 1 Header 2
Data 1 Data 2 
    <table border="1">
      <tr>
        <th>Header 1</th>
        <th>Header 2</th>
      </tr>
      <tr>
        <td>Data 1</td>
        <td>Data 2</td>
      </tr>
    </table>

Lesson 6: Forms and Input Elements

Creating Forms in HTML:
Forms are used to collect user input. The <form> tag wraps form elements like text fields, radio buttons, checkboxes, and submit buttons.

Example (Form):







    <form action="/submit" method="POST">
      <label for="name">Your Name:</label>
      <input type="text" id="name" name="name"><br><br>

      <label for="email">Your Email:</label>
      <input type="email" id="email" name="email"><br><br>

      <label for="message">Message:</label>
      <textarea id="message" name="message"></textarea><br><br>

      <input type="submit" value="Submit">
    </form>

Lesson 7: HTML Attributes

What are HTML Attributes?
Attributes provide additional information about an element. They are always placed within the opening tag and generally come in name-value pairs.

Example (Attributes):

Here are some examples of HTML elements with attributes:

      <a href="https://www.example.com" id="link1" class="primary">Click here</a>
      <img src="https://via.placeholder.com/150" alt="Example Image" width="150" height="150">

Result:

Click here

Here, the <a> tag has an href attribute specifying the URL, an id attribute for identification, and a class for styling. The <img> tag has src, alt, and size attributes.

Lesson 8: HTML Semantic Elements

What Are Semantic Elements?
Semantic HTML refers to elements that have meaning and describe the type of content they contain. Using semantic tags improves accessibility and search engine optimization (SEO).

Example (Semantic Elements):

Here’s an example of using semantic HTML elements:

      <header>
        <h1>Website Header</h1>
      </header>
      <article>
        <h2>Blog Post Title</h2>
        <p>This is a blog post...</p>
      </article>
      <footer>
        <p>© 2025 My Website</p>
      </footer>

Result:

Website Header

Blog Post Title

This is a blog post...

© 2025 My Website

In this example, we use semantic elements such as <header>, <article>, and <footer> to structure the page. These elements help describe the role of the content they contain.
